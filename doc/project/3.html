    # 需求说明

    只支持中英文与数字输入

    # 实现思路

    思路很简单，监听input 事件，正则匹配非法字符将其过滤， 一开始的代码是这样的<input :value="extensionValue" @input="filterExtensionValue"/>，但会有个问题，切换成中文输入法的时候在输拼音的过程中也会被input事件捕获到，导致出现异常的结果。后来改成v-model的形式解决了问题（代码如下），猜想是vue 框架在 v-model 中对input 做了处理，导致其过滤中文输入拼音的请情况，有待进一步研究。

    # 解决方案

      <input 
    class="ext-input" 
    placeholder="输入扩展值（支持中英文和数字）" 
    v-model="extensionValue" 
    @input="filterExtensionValue"
    maxlength="100"
  />

  filterExtensionValue(e) {
    // 只支持汉字、英文、数字
    this.extensionValue = e.target.value.replace(/[^0-9a-zA-Z\u4e00-\u9fa5]/g, '')
  }
